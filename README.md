## üíª Our Technology Stack

We operate with a **modern, asynchronous, and performance-oriented** technology stack, designed to adapt to the dynamic workflows of a multinational holding company.

### Main Stack (MERN/V)

| Category | Technology | Primary Use |
| :---: | :--- | :--- |
| **Frontend** | **React & Vue.js** | Development of dynamic and highly reactive user interfaces. |
| **Language** | **TypeScript** | Ensures robustness, scalability, and early error detection in the codebase. |
| **Backend** | **Node.js** | Fast and efficient runtime environment for API services. |
| **Framework** | **Express.js** | Building robust and secure RESTful APIs. |

### Architecture

We embrace a **Microfrontends** architecture.

* **Philosophy:** Breaking down monolithic applications into smaller, independent components.
* **Advantage:** Allows our teams to work **autonomously, deploy quickly**, and maintain **high adaptability** to the constant regulatory and financial changes of the holding company.
